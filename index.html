This is an application for testing the behavior of JavaScript framework, such as Angular Ember, for seriously big tables including thousands of records.

Firstly, choose a framework for which you would like to run the tests, and then choose a test. There are two of them: scroll test and speed test.

Important! For the proper results, each run test will be opened in a new tab. And, before running the test, please, make sure that tabs from previous test are closed. To run several tests in parallel is not recommended due to limitations of browsers performance.
